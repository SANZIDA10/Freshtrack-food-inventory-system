# Lab Topic Coverage for FreshTrack

This document maps the listed lab topics to the SQL and PL/SQL files in the project.

## Covered topics
- Lab 01 SQL: basic SQL queries and data retrieval in sql/03_queries.sql and sql/06_lab_topic_examples.sql
- Lab 02 DDL and DML: table creation and sample data in sql/01_create_tables.sql and sql/02_insert_data.sql
- Lab 03 Constraints: primary keys, foreign keys, unique constraints, check constraints, and defaults in sql/01_create_tables.sql
- Lab 04 SELECT and aggregate functions: SELECT, GROUP BY, HAVING, ORDER BY, SUM, COUNT in sql/03_queries.sql and sql/06_lab_topic_examples.sql
- Lab 05 Subquery, set operations, and views: subqueries, UNION, and views in sql/03_queries.sql, sql/04_views.sql, and sql/06_lab_topic_examples.sql
- Lab 06-1 Join operation: INNER JOIN examples in sql/03_queries.sql
- Lab 06-2 Outer Join: LEFT/RIGHT/FULL OUTER JOIN examples in sql/06_lab_topic_examples.sql
- Introduction to PLSQL: PL/SQL blocks, exceptions, procedures, functions, cursors, and triggers in sql/05_procedures_triggers.sql and sql/06_lab_topic_examples.sql
- PL/PLSQL presentation content: project documentation and SQL examples aligned with the lab material in README.md and sql/06_lab_topic_examples.sql
